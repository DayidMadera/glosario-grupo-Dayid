## Terminos de Hardware
### CPU — Unidad Central de Procesamiento
Es el cerebro del computador. Procesa todas las instrucciones
que le damos al sistema. Sin el, ninguna operacion es posible.


### RAM — Memoria de Acceso Aleatorio
Es la memoria temporal del computador. Guarda lo que estas
usando en este momento. Al apagar el computador, se borra.



### SSD — Disco de Estado Solido
Es donde se guarda la informacion de forma permanente.
Es mas rapido que un disco duro tradicional (HDD).

## Terminos de Software

##  Software

## 'Software de Sistema': Es la base que permite el funcionamiento del hardware y gestiona recursos.
Sistema Operativo (SO): Gestiona el hardware y servicios (Windows, Linux, Android).

## 'Controladores de Dispositivo (Drivers)': Conectan el sistema operativo con el hardware.
BIOS/Firmware: Programa inicial de arranque.

## 'Software de Aplicación': Programas diseñados para que el usuario final realice tareas.
Aplicaciones de ofimática (Office), navegadores web, reproductores multimedia, redes sociales.

## Terminos Generales de Programacion

### Algoritmo

Un algoritmo es un conjunto de pasos ordenados que se utilizan para resolver un problema o realizar una tarea.

### Variable

Una variable es un espacio en memoria donde se almacena un valor que puede cambiar durante la ejecución del programa.

Ejemplo:
edad = 20

### Constante

Una constante es un valor que no cambia durante la ejecución del programa.

Ejemplo:
PI = 3.1416
